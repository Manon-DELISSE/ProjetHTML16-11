    • Baten Kaitos : Les Ailes éternelles et l’Océan perdu est un jeu de rôle pour GameCube, sorti en 2003 au Japon et 2005 en Europe. Le jeu se déroule dans un univers de type fantasy, où des îles flottent dans le Ciel. L’histoire se passe dans un monde d’îles flottant dans le Ciel. Mille ans avant le début du jeu, un dieu maléfique nommé Malpercio envahit le monde. Il fut défait par cinq héros magiciens, qui scellèrent le dieu maléfique dans les cinq « Magnus ultimes », chacun caché dans une des cinq îles principales. Les magiciens quittèrent la Terre polluée et firent léviter les îles dans le Ciel. Les humains se virent ensuite pousser des « Ailes du cœur ».

    Si vous cherchez la solution complète : Le coin de Vally, pour finir le jeu à 100%
